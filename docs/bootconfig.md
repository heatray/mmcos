# Boot Config

Config location: `MicroMachines_Data/StreamingAssets/BootConfig_User.xml`

Default content:

```xml
<?xml version="1.0" encoding="utf-8"?>
<BootConfig>
  <property name="m_serverEnvironment" value="CDN" />
</BootConfig>
```
